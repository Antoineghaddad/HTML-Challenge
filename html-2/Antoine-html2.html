Web Development

Web development refers to building, creating, and an maintaining websites.
It includes aspects such as web design, web publishing, web programming,
and database management.

    Languages Needed:
  • HTML
  • CSS
  • javascript
  • nodejs
  • reactjs

JavaScript is commonly used in web browsers. Web development tools commonly
include a panel to debug scripts by allowing developers to add watch expressions,
breakpoints, view the call stack, and pause, step over, step into, and step out of
functions while debugging JavaScript.
