GitHub is a web-based version-control and collaboration platform
for software developers. Microsoft, the biggest single contributor to
GitHub, initiated an acquisition GitHub for $7.5 billion in June, 2018. GitHub,
which is delivered through a software-as-a-service (SaaS) business model,
was started in 2008 and was founded on Git, an open source code management
system created by Linus Torvalds to make software builds faster.

Because GitHub is so intuitive to use and its version-control tools are so useful
for collaboration, nonprogrammers have also begun to use GitHub to work on document-based
and multimedia projects. GitLab is an open source alternative to GitHub.
